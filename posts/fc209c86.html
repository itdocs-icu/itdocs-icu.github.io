1.安装

wget https://github.com/nvm-sh/nvm/archive/refs/tags/v0.38.0.tar.gz

2.解压

mkdir -p /root/.nvm
tar -zxvf v0.38.0.tar.gz -C /root/.nvm

3.环境配置

编辑~/.bashrc

vim ~/.bashrc 


export NVM_DIR="$HOME/.nvm/nvm-0.38.0" # 注意位置是不是一致的
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"

4.加载配置

source ~/.bashrc

5.安装node

nvm install 12

注：系统原有的node将失效

6.使用说明

  • 1.查看nvm安装的 node——nvm ls
  • 2.卸载对应的 node——nvm uninstall xxx
  • 3.切换 node——nvm use xxx

7.gcc解决

yum install "gcc-c++.x86_64"
